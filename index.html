Về Tôi

Xin chào! Tôi là một người đam mê công nghệ và thích giải quyết các vấn đề thông qua mã nguồn, với một tình yêu đặc biệt dành cho việc khám phá những "Atlantis" dữ liệu chưa được tìm thấy. Giống như một nhà thám hiểm dưới đại dương sâu thẳm, tôi tin vào việc xây dựng những giải pháp mạnh mẽ, sạch sẽ và mượt mà.

Với tư duy logic và sự sáng tạo, tôi luôn tìm cách để biến những ý tưởng phức tạp thành những sản phẩm dễ sử dụng và tối ưu nhất.

Kỹ Năng

HTML5 / CSS3 JavaScript (ES6+) Python / Django SQL / PostgreSQL Git / GitHub AI / Machine Learning (Cơ bản)

Dự Án

Thư viện Atlantis
